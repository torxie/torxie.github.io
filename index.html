hexo

发表于 2017-09-17 | 分类于 veekocnn

hexo 基于nodejs生成的静态博客，相较于jekyII部署到github好处在于更多的插件支持。hexo编辑完新文章后，需要 hexo genereate 和 hexo deploy 自动部署到github方便快捷。

安装

  1. nodejs
  2. git
  • nodejs安装完毕之后：win + r启动cmd，输入node，执行console.log(“hello, boy”)，可验证是否安装成功
  • 设置npm（nodejs的安装管理工具）的缓存和安装模块路径
    • npm config set prefix “D:\Program Files\nodejs\node_global”
    • npm config set cache “D:\Program Files\nodejs\node_cache”
    • 系统变量中加上NODE_PATH，路径为D:\Program Files\nodejs\node_modules\
  • 通过npm安装hexo
    • npm install -g hexo-cli
    • 系统变量PATH中，加入hexo的启动（D:\Program Files\nodejs\node_global\node_modules\hexo-cli\bin）
  • 验证git客户端和github的连接,因为hexo部署目前使用ssh连接。启动 ## git bash
    • 设置Git的user name和email：
      • $ git config –global user.name “torxie”
      • $ git config –global user.email “torxie@outlook.com”
    • 生成sshkey：
      • $ ssh-keygen -t rsa -C “torxie@outlook.com”
      • 将生成的秘钥id_rsa.pub内容粘贴到github里面的ssh key（理解为该github允许这台pc ssh连接）
    • ssh git@github.com 验证ssh连接是否成功

开始hexo启动、部署之旅

  1. 在文件管理系统中新建一个文件夹，用于放置hexo博客。右键启动git bash；
  2. hexo init，博客默认代码生成
  3. npm install
  4. hexo server，启动本地服务器，localhost：4000中可查看生成的hexo模块

  5. git 部署

    • npm install hexo-deployer-git –save （设置hexo的git组件）
    • 修改_config.yml文件，设置与github repository的关联
      type: git
repo: git@github.com:torxie/torxie.github.io.git
branch: master
    • hexo genereate，生成静态博客文件

    • hexo deploy，将静态博客文件push到git repository中。

    • npm代理设置：
      npm config set proxy http://server:port
      npm config set https-proxy http://server:port
      npm config set proxy http://username:password@server:port
      npm config set https-proxy http://username:pawword@server:port

  • git代理设置
    git config –global http.proxy http://username:password@IP:port
    git config –global https.proxy http://username:password@IP:port

  • 内网（under network）
    git config –global url.”https://".insteadOf git://

跳闸

发表于 2017-09-17 | 分类于 veekocnn

星辰高照，马路边地铁施工声透过水泥墙隐隐在耳边轰鸣。突然被旁边的插线板一生轻微爆响，将睡梦中的我们惊醒。

只见插线板火花一闪，空调停止给这灼热的夜晚降温。初以为小楼跳闸然后突然来电，导致全屋电路烧毁。但万幸的是厕所的灯可以亮起，为查找出元凶提供了线索–跳闸！

出去把电闸推上去，veeko找到了元凶。手机充电器烧了，真是不幸中的大幸，没有伤到人以及造成燃烧。
